ClientFuse Text Logo

To download as PNG:

  1. Right-click on the logo text above
  2. Select "Save image as..." or "Copy image"
  3. Alternatively, take a screenshot and crop to the logo
  4. For higher quality, use browser DevTools to set a larger font size (e.g., 96px or 128px) and export

The background is transparent. The gradient goes from blue (#2563eb) to purple (#9333ea).
